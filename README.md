# Arjit Chauhan - Personal Portfolio

A professional, minimalist portfolio website for **Arjit Chauhan**, Senior Automotive Systems Engineer at Qualcomm.

## 🚀 Overview

This repository contains the source code for my personal portfolio, showcasing my work in Automotive SoC performance, on-device AI enablement, and systems benchmarking. The design is built to be clean, professional, and technically focused.

## 🛠️ Tech Stack

- **Core:** Semantic HTML5
- **Styling:** Vanilla CSS (Custom design system)
- **Icons:** Font Awesome
- **Deployment:** GitHub Pages

## 📂 Project Structure

- `index.html`: Main portfolio structure and content.
- `css/custom.css`: Custom design tokens, typography, and responsive layout logic.
- `js/`: Scripts for interactive elements (navigation, scroll effects).
- `images/`: Portfolio assets including profile pictures and background images.

## 🌟 Key Sections

- **Hero:** Quick introduction and key role at Qualcomm.
- **About:** Deep dive into professional background and technical interests.
- **Focus Areas:** Technical specialization in SoC Performance, On-Device AI, and Concurrent Workloads.
- **Experience:** Detailed career timeline and educational background.
- **Projects:** Selected works including automation frameworks and optimization projects.

### 🤖 AI-Augmented Workflows

A sub-focus on leveraging agentic AI to accelerate research, visualization, and specialized tooling.

- **[Thought](https://arjitc12.github.io/Thought/):** An experimental exploration into cognitive patterns and history, visualized through custom 3D and 2D interfaces.
